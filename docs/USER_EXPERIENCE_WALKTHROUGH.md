### Before (CLI):
1. Install Python dependencies (confusing for non-devs)
2. Navigate to correct directory in terminal
3. Type command with exact syntax
4. Wait with no feedback
5. Get 10+ separate files
6. Open each file individually
7. Try to mentally connect insights across files
8. Copy data into other tools for visualization
9. Hard to share or explore

**Time to insights: 30-60 minutes**
**Accessibility: Technical users only**
**Friction: High**

### After (Web App):
1. Double-click app icon
2. Drag and drop file
3. Watch progress in real-time
4. See insights immediately
5. Explore interactively
6. Click to drill down
7. Export what you need
8. Share safely

**Time to insights: 3-5 minutes**
**Accessibility: Everyone**
**Friction: Minimal**
